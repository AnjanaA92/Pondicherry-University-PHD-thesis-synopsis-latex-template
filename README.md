# Pondicherry-University-PHD-synopsis-LATEX-template
Synopsis Template
This repository contains a LaTeX-based synopsis template designed specifically for PhD scholars. It provides a structured and easy-to-edit format for preparing research synopses in a clear, professional, and organized manner.


Overview

The template is created to simplify the process of writing a PhD synopsis by offering predefined sections, formatting styles, and examples. Users can customize the content according to their research requirements without worrying about formatting issues.

Features

Well-structured LaTeX format for synopsis writing
Predefined sections for easy editing
Support for tables, figures, equations, and bullet points
Clean and professional layout
Easily customizable for any research domain

How to Use

Download or clone the repository
Open the .tex file in any LaTeX editor (Overleaf, TeXstudio, etc.)
Replace the placeholder content with your own research details
Compile the document to generate the final PDF

Template Includes

Title page format
Chapter-wise synopsis structure
Sample tables (normal and sideways)
Figure insertion format
Equation formatting
Bullet point structure
